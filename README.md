# widget-ping-pong-game
An interactive ping-pong game built with React and Canvas API. Perfect for embedding in websites as a fun, engaging widget.

![example image](/example/example.png)

## Development

### Install dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build static files

```bash
npm run build
```

### Deploy GitHub Page
```bash
npm run deploy
```
